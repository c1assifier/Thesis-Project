# Дипломная работа

Этот проект является дипломной работой по теме адаптивного обучения Python. Система представляет собой full-stack веб-приложение, в котором студент проходит вводный модуль, диагностику, изучает теорию по урокам и решает практические задания, а платформа подстраивает следующий шаг обучения под текущий уровень пользователя.

## Технологии

- Backend: FastAPI, SQLAlchemy 2, Alembic, PostgreSQL
- Frontend: React 18, TypeScript, Vite, React Router, React Query, Axios, Monaco Editor, Tailwind CSS
- Инфраструктура: Docker Compose

## Логика курса

Курс разбит на уровни `Course -> Module -> Lesson -> Topic -> Theory -> Exercise`. Такая структура позволяет делить материал на небольшие учебные блоки: сначала студент проходит модуль, внутри него изучает уроки, затем конкретные темы и после этого закрепляет материал упражнениями.

После регистрации пользователь проходит вводный этап и диагностику. На основе результатов и последующих решений система отслеживает прогресс, ошибки, количество попыток и выбирает, какой модуль или шаг стоит показать дальше.

## Подсказки и адаптация

Подсказки используются как часть адаптивной логики, а не как отдельная статическая справка. Если студент не справляется с задачей, система анализирует попытки решения, может предложить более простое объяснение темы, подсказку по текущему упражнению и направить на следующий подходящий шаг обучения.
